## HELP

Arguments: 

  -v - Verbose
  
  -h - Displayes this message
  
  -t - Set amout of threads to run
  
  -ip - Set victum ip
  
  -port - Set victum port
  
  -passlist - Set a password list to use
  
  -username - Set a username list to use
